# K-Means Clustering Study Guide

## Introduction
K-Means is an unsupervised machine learning algorithm used to group similar data points into clusters.

## What is Clustering?
Clustering is the task of grouping similar data without labels.

## How K-Means Works
1. Choose number of clusters (K)
2. Initialize centroids randomly
3. Assign each point to nearest centroid
4. Update centroid positions
5. Repeat until convergence

## Algorithm Steps
- Select K
- Initialize centroids
- Assign points to closest centroid
- Update centroids
- Repeat until stable

## Distance Formula
Euclidean distance:
√((x2 - x1)² + (y2 - y1)²)

## Elbow Method
Used to find optimal K by plotting WCSS vs K.

## Advantages
- Simple
- Fast
- Works well on small datasets

## Disadvantages
- Need to choose K manually
- Sensitive to outliers

## Applications
- Customer segmentation
- Image compression
- Pattern recognition
